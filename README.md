# Binary Counter LEDs_stm32f401ccu6

## Overview

A bare-metal STM32 project demonstrating a 4-bit binary counter using four LEDs on the STM32F401CCU6 Black Pill board.

## Hardware

* STM32F401CCU6 Black Pill
* 4 LEDs
* 4 × 220Ω Resistors
* ST-Link V2
* STM32CubeIDE

## Features

* GPIO Output Configuration
* Bitwise Operations
* Binary Number Representation
* Multiple GPIO Output Control
* Bare-Metal Programming (No HAL)

## Program Flow

```text
0000
 ↓
0001
 ↓
0010
 ↓
...
 ↓
1111
 ↓
Repeat
```

## Expected Output

The four LEDs count in binary from **0000** to **1111** (0–15) continuously.
